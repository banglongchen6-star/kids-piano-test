# 荒唐节拍社

使用 20 张原创双格概念图制作的 PartyKeys 36 节奏游戏原型。

## 演出结构

每关固定为两个连续四拍：

1. **预告四拍**：显示关卡名称与随机和弦，PartyKeys 对应按键持续亮灯；画面依次显示“记住、3、2、1”。
2. **表演四拍**：灯光熄灭，双格概念图按 `第一帧 → 第二帧 → 第一帧 → 第二帧` 每拍切换一次。玩家需要在四个拍点分别弹奏刚才记住的和弦。

20 个场景连续演出，音乐全程不停顿。

## 键盘连接

进入游戏时不显示任何关卡画面。玩家允许 Web MIDI + SysEx 后：

1. 检查 MIDI 输入与输出。
2. 通过 PartyKeys 官方 SysEx 协议让 36 键全部亮绿灯一秒。
3. 自动熄灭全部灯光并显示连接成功页。
4. 玩家主动点击“开始游戏”后音乐与第一关预告才会开始。

建议使用最新版 Chrome 或 Edge。

## 无设备调试

游戏判定保留电脑键盘映射：

- `A`：C
- `S`：Dm
- `D`：Em
- `F`：F
- `G`：G
- `H`：Am

正式流程仍要求先完成 PartyKeys 连接确认。
